ATOM WebAR package
==================

Files included:
- index.html      : WebAR page (open this on your phone)
- cube.png        : product image for Cube hologram
- jerry.png       : product image for Jerry Can hologram
- targets.mind    : MindAR image target file (contains both markers)

How to deploy to GitHub Pages (quick):
1. Go to your repo: https://github.com/<your-username>/<your-username>.github.io
2. Click 'Add file' -> 'Upload files'
3. Drag & drop ALL files from this ZIP into the upload area (index.html, cube.png, jerry.png, targets.mind)
4. Commit changes
5. Wait ~30 seconds, then open: https://<your-username>.github.io/

Notes:
- The first image in the targets.mind is targetIndex 0 (Cube). The second is targetIndex 1 (Jerry Can).
